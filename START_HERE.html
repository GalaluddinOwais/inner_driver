⚠️ Before You Start - Checklist

  • Python dependencies installed: pip install -r requirements.txt
  • Database migrated: python manage.py migrate
  • .env file created (copy from .env.example)
  • At least one VehicleType exists in database
  • Django server running: python manage.py runserver
  • Postman collection imported

🔄 Sample Workflow (15 Minutes)

  1. Register Driver (Postman) → OTP email sent ✅
  2. Recharge Balance (Postman) → $500 added ✅
  3. Set Available (Postman) → $50 deducted ✅
  4. Register Rider (Postman) → Token saved ✅
  5. Request Ride (Postman as Rider) → Ride created ✅
  6. Connect WebSocket (Browser) → See notification 🔔
  7. Accept Ride (Postman as Driver) → Rider notified ✅
  8. Send Location (Browser) → Rider sees updates 📍
