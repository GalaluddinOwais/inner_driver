🐛 COMMON ISSUES & FIXES:
================================================================================

"Connection refused"
   → Django server not running
   → Run: python manage.py runserver

"Unauthorized 401"
   → Token expired
   → Use Refresh Token or Login again

"Forbidden 403"
   → Wrong user type or not your resource
   → Check you're logged in as correct user

"WebSocket connection fails"
   → Invalid token or token expired
   → Get fresh token from login

"Email not sending"
   → .env not configured
   → Use console backend for testing:
      EMAIL_BACKEND = 'django.core.mail.backends.console.EmailBackend'
